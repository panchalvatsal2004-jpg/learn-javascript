JavaScript Ki History (History of JavaScript)

1.JavaScript kab bana?
JavaScript ko 1995 mein banaya gaya tha.

2.JavaScript ko kisne banaya?
Is language ko Brendan Eich ne banaya tha, jo Netscape company mein kaam karta tha.

3.JavaScript ka purpose kya tha?
Us waqt web pages bahut simple the, bas text aur images hote the. Netscape ne chaha ki web pages thode interactive ho jayein, jaise buttons par click karne se kuch ho, forms fill karne par validation ho, bina page reload kiye. Isliye JavaScript bani.

4.Pehla naam kya tha?
JavaScript ka asli naam Mocha tha, phir ise LiveScript kaha gaya, aur finally JavaScript naam diya gaya.

5.JavaScript aur Java me farak?
Naam similar hai, lekin dono alag alag languages hain. Java ek full programming language thi, aur JavaScript browser ke liye ek scripting language banayi gayi thi.

6.JavaScript ko standardize kaise kiya gaya?
1997 mein, JavaScript ko ECMAScript standard ke under laya gaya, jisse language ko har browser mein same tarah chalaya ja sake.

7.Aaj JavaScript
Aaj JavaScript web development ki sabse popular language hai, sirf browser mein nahi, server side (Node.js) aur mobile apps, games, etc. mein bhi use hoti hai.
